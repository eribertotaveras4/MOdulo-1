  1. Definicion de CSS.

    2. Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado para describir la presentación de un documento escrito en HTML.


  2. Funcionaminento Basico de CSS.

    3. El funcionamiento básico de CSS se refiere a cómo se aplican las reglas de estilo a los documentos HTML para modificar su apariencia visual.


  3. Incluir CSS en un archivo HTML.

    4. Para incluir CSS en un archivo HTML tenemos que utilizar una etiqueta para linkear y combinar estos dos lenguajes llamada (LINK)


  4. Definicion de una regla CSS.

        5. Una regla CSS. es una instrucción que define cómo se deben presentar ciertos elementos HTML en una página web.

  5. Elemento de una regla CSS.
    • Regla

      • Es una instrucción completa en CSS que define cómo se debe estilizar un elemento HTML

    • Selector

      • Es la parte de la regla que indica a qué elementos HTML se aplicará el estilo

    • Bloque de declaracion

      • Es la parte de la regla que contiene las propiedades y valores que se aplicarán al elemento seleccionado.

    • Propiedad

      • Es una característica del elemento HTML que quieres cambiar, como el color del texto o el tamaño de la fuente

    • Valor

      • Especifica cómo quieres cambiar la propiedad. Por ejemplo, blue para el color o 16px para el tamaño de la fuente.

  6. Medios CSS.

    7. Los medios CSS son técnicas que permiten aplicar estilos específicos a diferentes dispositivos y condiciones de visualización.

  7. Comentarios CSS

    8. son líneas o bloques de texto que se incluyen en el código CSS para proporcionar información adicional o notas explicativas (/* comentario */)

  8. Sintaxis de la definición de las propiedades CSS.

    9. Consiste en escribir reglas que aplican estilos a elementos HTML. Cada regla tiene un selector que indica a qué elementos se aplicará el estilo.
